# vue3-webpack5-template

## 简介
基于`webpack5`搭建的一个vue3模板，支持：
- [x]  scss/postcss
- [x]  port检查
- [x]  vue3
- [x]  友好输出
- [x]  环境变量导入
- [x]  打包缓存
